# Privacy Policy

Effective date: 2026-06-24

Animal Chess is a local-first Windows desktop game developed by BabyQ.

## Summary

Animal Chess does not require an account, does not collect personal information, and does not upload gameplay data in the current version.

## Data Collection

The current version of Animal Chess does not collect:

- Names, email addresses, phone numbers, or account identifiers
- Gameplay history
- Device identifiers
- Location data
- Contacts
- Files or documents from your device
- Screenshots, screen recordings, or microphone input
- Keyboard input outside the game
- Browser history or website URLs
- Telemetry or analytics events

## Local Data

The current development version may keep runtime state in memory while the app is open, such as:

- Current board state
- Selected language
- Selected theme
- Current match progress

At this stage, these settings are not uploaded to any server.

Future versions may add local settings persistence. If added, the data will be stored locally on your device and used only to restore your preferences.

## Network Access

The game includes a button that opens the illustrated rules website:

<https://LittleBabyQ.github.io/AnimalChess/>

Opening this link uses your default web browser. The website is hosted by GitHub Pages and may be subject to GitHub's own privacy practices.

The game itself does not send gameplay data to this website.

## Third-Party Services

The current version does not integrate third-party analytics, advertising SDKs, cloud save services, or user tracking systems.

## Children's Privacy

Animal Chess is a casual board game and does not knowingly collect personal information from children.

## Changes

This privacy policy may be updated when new features are added. Material changes will be reflected in this file.

## Contact

For privacy-related questions, please contact the project owner through the GitHub repository or Microsoft Store listing once available.
